### 🌟 Featured Projects

**🔹 SmartSure — Production-Grade Microservices Insurance Platform** *(Mar 2026 – May 2026)*
> **Stack:** .NET 10, ASP.NET Core, Angular 17, SQL Server, RabbitMQ, JWT, EF Core, SignalR, xUnit
- Architected 5 independent microservices (Auth, Policy, Claim, Admin, Notification) using Clean Architecture (API → Application → Domain → Infrastructure)
- Built event-driven communication via RabbitMQ with domain events for zero-coupled, cascading operations
- Implemented JWT-based RBAC, global exception middleware, and `/health` endpoints across all services
- Engineered an Angular 17 frontend with Standalone Components, Signals, lazy loading, and route guards
- Developed a real-time SignalR admin dashboard for live revenue, user, and claims analytics
- Achieved full unit test coverage with xUnit, Moq, and Coverlet for coverage reporting
